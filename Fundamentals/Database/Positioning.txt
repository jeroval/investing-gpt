
RÉFÉRENTIEL DE POSITIONNEMENT RELATIF ET DE BENCHMARKING FONDAMENTAL
Objet du document
Les référentiels Adaptatif, Systémique et All Weather permettent d'évaluer la qualité fondamentale d'une entreprise.
Ce document poursuit un objectif différent.
Il vise à déterminer où se situe une entreprise avant de l'évaluer.
Une entreprise ne doit jamais être jugée dans l'absolu.
Elle doit être positionnée relativement :
à son marché ;
à son secteur ;
à son indice de référence ;
aux autres entreprises de qualité ;
aux référentiels fondamentaux.
Ce document constitue la couche de mesure utilisée avant toute analyse.
Principe Fondamental
Toute analyse doit commencer par une question simple :
Par rapport à quoi cette entreprise est-elle évaluée ?
Les termes :
excellente ;
résiliente ;
dominante ;
rentable ;
exceptionnelle ;
sont interdits tant que le référentiel de comparaison n'est pas précisé.
PILIER 1 — Univers de Comparaison
Identifier la fonction économique principale.
Exemples :
logiciel ;
infrastructure ;
paiement ;
santé ;
industrie ;
télécommunication ;
consommation ;
énergie ;
assurance ;
services financiers.
L'entreprise est d'abord comparée à ses pairs naturels.
PILIER 2 — Position Concurrentielle
Déterminer la place de l'entreprise dans son marché.
Classification :
🏆 Leader
🟢 Co-Leader
🔵 Challenger
🟡 Acteur Moyen
🟠 Niche Spécialisée
🔴 Retardataire
Question :
Quelle serait la place naturelle de cette entreprise dans son écosystème ?
PILIER 3 — Position Sectorielle
Comparer l'entreprise à son secteur.
Mesures possibles :
croissance ;
marge ;
rentabilité ;
cash-flow ;
retour sur capital ;
bilan ;
création de valeur.
Position :
Top 10 %
Top 25 %
Moyenne Supérieure
Moyenne
Moyenne Inférieure
Bottom 25 %
Bottom 10 %
PILIER 4 — Position Mondiale
L'entreprise est-elle :
mondiale ?
régionale ?
nationale ?
locale ?
Une entreprise dominante dans son pays peut rester moyenne à l'échelle mondiale.
Cette distinction est obligatoire.
PILIER 5 — Comparaison à un Indice
Question obligatoire :
Pourquoi détenir cette entreprise plutôt qu'un indice ?
Références possibles :
MSCI World
Nasdaq-100
CAC 40
indice sectoriel pertinent
L'entreprise doit justifier sa présence face à une alternative passive.

Évaluation :
Exceptionnel
Supérieur
Solide
Moyen
Faible
Question :
Cette organisation est-elle capable de se réinventer lorsque son environnement change ?
PILIER 7 — Positionnement Systémique
Évaluation :
Exceptionnel
Supérieur
Solide
Moyen
Faible
Question :
Le système économique dépend-il réellement de cette entreprise ?
PILIER 8 — Positionnement All Weather
Évaluation :
Exceptionnel
Supérieur
Solide
Moyen
Faible
Question :
L'entreprise reste-t-elle robuste dans différents régimes économiques ?
PILIER 9 — Carte d'Identité Relative
Toute analyse débute obligatoirement par :
Position Concurrentielle
Position Sectorielle
Position Mondiale
Position vs Indice
Position Adaptative
Position Systémique
Position All Weather
Cette carte doit pouvoir être comprise en moins de trente secondes.
PILIER 10 — ÉCHELLE DE MATURITÉ FONDAMENTALE
Objectif :
Situer instantanément l'entreprise dans l'univers global des sociétés cotées.
Cette classification synthétise tous les autres piliers.
🔴 Classe E — Fragile
Modèle économique vulnérable.
Dépendances fortes.
Avantages concurrentiels limités.
Visibilité réduite.
Risque élevé d'obsolescence ou de destruction de valeur.
🟠 Classe D — Correcte
Entreprise viable.
Position concurrentielle acceptable.
Mais peu d'avantages durables.
Création de valeur incertaine à long terme.
🟡 Classe C — Solide
Entreprise de qualité.
Bonne rentabilité.
Position concurrentielle défendable.
Résilience satisfaisante.
Capable de traverser plusieurs cycles.
🟢 Classe B — Excellente
Leader ou co-leader.
Avantages concurrentiels importants.
Bonne allocation du capital.
Résilience démontrée.
Création de valeur durable.
Entreprise susceptible d'être détenue pendant plusieurs décennies.
🏆 Classe A — Référence Mondiale
Organisation exceptionnelle.
Leadership durable.
Capacité d'adaptation démontrée.
Position stratégique forte.
Résilience exceptionnelle.
Allocation du capital remarquable.
Probabilité élevée de rester pertinente sur plusieurs générations.
Ces entreprises constituent les références de leur univers économique.
Question Ultime
Avant toute conclusion, répondre à la question :
Cette entreprise paraît-elle exceptionnelle parce qu'elle est réellement exceptionnelle, ou parce que son référentiel de comparaison est faible ?
Si le référentiel change et que la conclusion change radicalement, alors l'analyse est incomplète.
Synthèse Finale
Le rôle de ce document n'est pas de sélectionner.
Le rôle de ce document est de situer.
Une entreprise doit toujours être :
Positionnée.
Comparée.
Contextualisée.
Puis seulement évaluée.
L'analyse commence par le positionnement.
Le jugement vient ensuite.
