# Études de cas : 10 formes d'organisations capables de recréer leur avenir

## Objectif

L'objectif n'est pas d'identifier les meilleures entreprises.

L'objectif est de comprendre les différents mécanismes permettant à une organisation de rester pertinente lorsque son environnement change.

Chaque étude de cas répond à la même question :

> Pourquoi cette organisation pourrait-elle encore être pertinente dans 50 ans ?

---

# Cas n°1 — Berkshire Hathaway

## Hypothèse initiale

Berkshire est-elle une compagnie d'assurance ?

## Analyse

Les assurances génèrent du capital.

Ce capital est ensuite réalloué vers :

* l'énergie ;
* le ferroviaire ;
* l'industrie ;
* les services ;
* les actions cotées.

La croissance ne dépend pas d'une activité particulière.

Elle dépend de la capacité de l'organisation à rediriger les ressources vers les meilleures opportunités disponibles.

## Diagnostic

L'avantage concurrentiel principal n'est pas un produit.

L'avantage concurrentiel principal est :

> la discipline d'allocation du capital.

## Pourquoi cette organisation pourrait survivre ?

Parce qu'elle peut remplacer ses moteurs économiques.

---

# Cas n°2 — Constellation Software

## Hypothèse initiale

Constellation est-elle une entreprise de logiciels ?

## Analyse

L'entreprise possède des centaines de logiciels.

Mais les logiciels changent constamment.

Le véritable point commun est ailleurs.

Chaque acquisition enrichit :

* les processus ;
* l'expérience ;
* les méthodes d'intégration ;
* les capacités d'évaluation.

## Diagnostic

Le véritable actif est :

> le système d'acquisition.

## Pourquoi cette organisation pourrait survivre ?

Parce qu'elle améliore continuellement sa capacité à acheter de nouvelles activités.

---

# Cas n°3 — Danaher

## Hypothèse initiale

Danaher est-elle une entreprise de santé ?

## Analyse

Au fil du temps :

* les sociétés détenues ont changé ;
* les secteurs ont changé ;
* les produits ont changé.

Ce qui est resté constant :

* le Danaher Business System.

Chaque acquisition est intégrée dans un système de management reproductible.

## Diagnostic

Le véritable actif est :

> le système opérationnel.

## Pourquoi cette organisation pourrait survivre ?

Parce qu'elle sait améliorer ce qu'elle achète.

## Diagnostic

* assureurs ;

## Diagnostic

Le véritable actif est :

> l'organisation du marché.

## Pourquoi cette organisation pourrait survivre ?

Parce qu'elle est devenue l'infrastructure du processus.

# Cas n°10 — Halma

## Hypothèse initiale

Halma est-elle une entreprise de sécurité ?

## Analyse

L'organisation regroupe des dizaines d'entreprises de niche.

Les activités sont différentes.

Les besoins fondamentaux sont similaires :

* sécurité ;
* conformité ;
* santé ;
* environnement.

## Diagnostic

Le véritable actif est :

> la composition disciplinée de niches critiques.

## Pourquoi cette organisation pourrait survivre ?

Parce qu'aucun moteur individuel n'est indispensable à la survie de l'ensemble.

Le but de ces études de cas n'est pas de conclure que ces entreprises sont les meilleures.

* amélioration opérationnelle ;

* diversification ;
* standardisation ;
* adaptation historique.
